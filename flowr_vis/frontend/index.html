Input Files

Two options for structure-based generation:

  1. Protein + Ligand — upload both to identify the binding site
  2. Protein Pocket only — upload a pre-cut pocket PDB and specify the number of heavy atoms to generate

Three options for ligand-based generation:

  1. Reference molecule — upload an SDF/MOL file
  2. SMILES string — enter a SMILES and generate a 3D conformer
  3. From scratch — specify only the number of heavy atoms (no reference needed)
